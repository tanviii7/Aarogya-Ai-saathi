# Aarogya AI Saathi

Aarogya AI Saathi is an AI-powered healthcare worker portal designed to simplify attendance management, daily task tracking, patient record management, and intelligent healthcare assistance. 
The platform combines modern web technologies with Artificial Intelligence to improve efficiency for frontline healthcare workers.

> **Disclaimer:** This project is developed for educational and demonstration purposes.

#  Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/aarogya-ai-saathi.git
```

Move into the project

```bash
cd aarogya-ai-saathi
```

Install dependencies

```bash
pip install -r requirements.txt
```

Create a `.env` file

```env
GEMINI_API_KEY=YOUR_API_KEY
```

Run the application

```bash
python main.py
```
